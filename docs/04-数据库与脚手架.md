## 2. 数据库设计

### 连接字符串

```
Server=(LocalDB)\MSSQLLocalDB;Database=LibrarySeatSystem;Trusted_Connection=True;MultipleActiveResultSets=true
```

### 表结构

#### Seat（座位表）

| 字段 | 类型 | 约束 |
|------|------|------|
| Id | int | PK, 自增 |
| Name | nvarchar(50) | 必填 |
| Location | nvarchar(100) | 必填 |
| HasPower | bit | 默认 false |
| Status | int | 默认 0 |
| CreatedAt | datetime2 | 默认当前时间 |

#### Reservation（预约表）

| 字段 | 类型 | 约束 |
|------|------|------|
| Id | int | PK, 自增 |
| SeatId | int | FK → Seat.Id |
| UserName | nvarchar(50) | 必填 |
| ReserveDate | datetime2 | 必填 |
| TimeSlot | nvarchar(20) | 必填 |
| Status | int | 默认 0 |
| CreatedAt | datetime2 | 默认当前时间 |

索引：`IX_Reservations_SeatId_ReserveDate_TimeSlot`（SeatId, ReserveDate, TimeSlot）

#### AdminUser（管理员表）

| 字段 | 类型 | 约束 |
|------|------|------|
| Id | int | PK, 自增 |
| Username | nvarchar(50) | 必填 |
| Password | nvarchar(100) | 必填（MD5 哈希） |

### 种子数据

- 座位：20 条（A-01 ~ D-05，含 1 条维护中）
- 管理员：1 条（admin / 123456）

## 4. 会话管理

- **体验账号**：`Session["CurrentUserName"]`，默认 "张三"，可在"我的预约"页切换
- **管理员登录**：`Session["AdminUsername"]`，登录成功后写入，未登录访问管理页自动跳转登录页
- Session 超时时间：30 分钟
