### 3.2 交互效果

| 交互 | 实现方式 |
|------|----------|
| 页面跳转 | 使用相对链接 |
| 弹窗显示/隐藏 | 使用 JavaScript toggle |
| 按钮禁用 | 使用 CSS disabled 状态 |
| 响应式切换 | 使用 CSS media query |

## 4. 视觉方向说明

### 4.1 整体视觉方向

**校园服务感 + 轻量后台感**

本系统定位为高校内部轻量工具，视觉上传达"可靠、简洁、易用"。

### 4.2 用户端界面风格

| 维度 | 风格 |
|------|------|
| 整体氛围 | 安静学习感，以蓝色+白色+灰色为主 |
| 布局方式 | 手机优先，垂直居中，卡片式列表 |
| 交互方式 | 简单直接，无花哨动画 |
| 参考对标 | 超星学习通、图书馆座位预约小程序 |

### 4.3 管理端界面风格

| 维度 | 风格 |
|------|------|
| 整体氛围 | 轻量后台感，干净利落 |
| 布局方式 | 电脑优先，表格布局，信息密度适中 |
| 交互方式 | 表格操作，弹窗表单 |
| 参考对标 | Ant Design Pro、Element Admin |

## 5. 共享组件与自定义样式

### 5.1 共享组件

| 组件 | 说明 |
|------|------|
| 顶部导航栏 | 深色背景，白色文字，当前页高亮 |
| 卡片 | 白色背景，灰色边框，8px 圆角 |
| 表格 | 斑马纹，悬停高亮，底部边框 |
| 按钮 | 主按钮（蓝色）、成功按钮（绿色）、危险按钮（红色）、描边按钮 |
| 状态标签 | 圆角标签，语义化颜色 |
| 弹窗 | 居中显示，遮罩层，确认/取消按钮 |
| 表单 | 统一高度，聚焦态，错误态 |

### 5.2 自定义样式文件位置

| 文件 | 路径 | 说明 |
|------|------|------|
| 主样式文件 | `prototype/static-v1/styles.css` | 包含所有自定义样式 |

### 5.3 样式特点

1. **CSS Variables**：使用 CSS 变量管理颜色、间距、圆角
2. **响应式设计**：手机优先，使用 media query 适配桌面端
3. **状态管理**：通过 CSS 类名控制按钮、输入框、标签的状态
4. **一致性**：所有组件使用统一的视觉规范
