```
结果：200 OK

```

### 测试 3：座位详情

```
请求：GET http://localhost:5000/Seat/Detail/1
结果：200 OK
验证：页面显示 5 个时段状态（可预约/已预约）
```

### 测试 4：预约提交

```
请求：POST http://localhost:5000/Reservation/Create/1
参数：SeatId=1, ReserveDate=2026-07-14, TimeSlot=14:00-16:00
结果：302 重定向到 /Reservation/My
验证：我的预约页显示 A-01 / 14:00-16:00 / 已预约
```

### 测试 5：冲突校验

```
请求：POST http://localhost:5000/Reservation/Create/1（重复提交）

验证：页面显示"该时段已被预约，请选择其他时段"
```

```
请求：GET http://localhost:5000/Reservation/My
```

### 测试 7：取消预约

```
请求：POST http://localhost:5000/Reservation/Cancel/{id}
```
