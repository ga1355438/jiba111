## 2. 变更摘要

### F-01：首页信息补全
- **文件**：`Views/Home/Index.cshtml`
- **变更**：添加当前用户（Session `CurrentUserName`，默认"张三"）和当前日期显示
- **样式**：`text-muted` + `<strong>` 标签，位于欢迎语和按钮之间

### F-02：我的预约页状态筛选
- **文件**：`DataAccess/ReservationRepository.cs`
- **变更**：`GetByUserAsync` 添加 `.Where(r => r.UserName == userName && (r.Status == 0 || r.Status == 2))`
- **效果**：只显示 Status=0（已预约）和 Status=2（已取消）记录，已签到(1)/已过期(3)/已签退(4)不再显示

### F-03：座位列表分页体验优化
- **文件**：`Views/Seat/List.cshtml`
- **变更**：分页导航增加"首页"和"末页"按钮，"上一页""下一页"在边界时自动 disabled

### F-04：预约日期选择器"今天"标签
- **文件**：`Views/Reservation/Create.cshtml`
- **变更**：日期选择器使用 `input-group`，右侧添加 `input-group-text` 显示"今天"

### F-05：统计页空数据友好提示
- **文件**：`Views/Admin/Statistics.cshtml`
- **变更**：热门座位和时段分布空数据时，显示引导性提示（如"预约座位后此处将显示 TOP5 统计"）

### F-06：管理端侧边栏小屏折叠
- **文件**：`Views/Shared/_AdminLayout.cshtml`
- **变更**：
  - 添加 `.sidebar-toggle` 按钮（仅 <768px 显示）
  - 添加 `.sidebar-overlay` 遮罩层
  - 侧边栏在小屏下默认隐藏，点击菜单按钮滑入
  - 导航链接点击后自动收起侧边栏
