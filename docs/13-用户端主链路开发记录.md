### 3.2 用户主链路走查

```
用户首页 (/)
  └─→ 座位列表 (/Seat/List?page=1)
        └─→ 座位详情 (/Seat/Detail/{id})
              └─→ 预约提交 (/Reservation/Create?seatId=&timeSlot=)
                    └─→ POST 冲突校验 → 写入 Reservation 表
                          └─→ 我的预约 (/Reservation/My) ← 可看到刚提交的预约
                                └─→ 取消预约 (POST /Reservation/Cancel/{id}) ← Status 0→2
```
