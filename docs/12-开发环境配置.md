## 2. 环境搭建步骤

### 2.1 安装 .NET SDK

```bash
# 下载并安装 .NET 6.0 SDK
# https://dotnet.microsoft.com/download/dotnet/6.0

# 验证安装
dotnet --version
```

### 2.2 安装 SQL Server LocalDB

```bash
# 安装 SQL Server Express LocalDB
# https://docs.microsoft.com/en-us/sql/database-configure-connection-string-aspnet-core

# 验证安装
sqllocaldb info
```

### 2.3 安装 EF Core 工具

```bash
dotnet tool install --global dotnet-ef --version 6.0.25
```

## 3. 项目配置

### 3.1 连接字符串

文件：`appsettings.json`

```json
{
  "ConnectionStrings": {
    "DefaultConnection": "Server=(localdb)\\mssqllocaldb;Database=LibrarySeatSystem;Trusted_Connection=True;TrustServerCertificate=True"
  }
}
```

### 3.2 项目文件

文件：`LibrarySeatSystem.csproj`

```xml
<Project Sdk="Microsoft.NET.Sdk.Web">
  <PropertyGroup>
    <TargetFramework>net6.0</TargetFramework>
  </PropertyGroup>
  <ItemGroup>
    <PackageReference Include="Microsoft.EntityFrameworkCore.SqlServer" Version="6.0.*" />
    <PackageReference Include="Microsoft.EntityFrameworkCore.Tools" Version="6.0.*">
      <PrivateAssets>all</PrivateAssets>
    </PackageReference>
  </ItemGroup>
</Project>
```

## 4. 运行方式

### 4.1 本地运行

```bash
cd LibrarySeatSystem
dotnet run --urls http://localhost:5000
```

访问：
- 用户端：http://localhost:5000
- 管理端：http://localhost:5000/Admin/Login

### 4.2 Docker 运行

```bash
cd LibrarySeatSystem
docker-compose up -d
```

访问：
- 用户端：http://localhost:5000
- 管理端：http://localhost:5000/Admin/Login

## 7. 常见问题

### 7.1 数据库连接失败

```
错误：A network-related or instance-specific error occurred
解决：
1. 确认 SQL Server LocalDB 已安装
2. 运行 sqllocaldb info 检查实例状态
3. 启动实例：sqllocaldb start
```

### 7.2 迁移失败

```
错误：No migration found
解决：
1. 删除 Migrations 目录
2. 重新创建迁移：dotnet ef migrations add InitialCreate
3. 重启应用
```

### 7.3 端口被占用

```
错误：Address already in use
解决：
1. 查找占用进程：netstat -ano | findstr :5000
2. 终止进程：taskkill /PID {进程ID} /F
3. 或更换端口：dotnet run --urls http://localhost:5001
```
