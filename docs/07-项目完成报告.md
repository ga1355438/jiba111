# 项目完成报告

> 项目名称：图书馆座位预约系统
> 完成日期：2026-07-14
> 技术栈：ASP.NET Core MVC (.NET 6) + Razor + SQL Server LocalDB + EF Core + Bootstrap

---

## 1. 项目概述

本项目是一个轻量级图书馆座位预约系统，分为用户端和管理端：
- **用户端**：面向学生，提供座位浏览、预约、查看和取消预约等功能
- **管理端**：面向图书馆管理员，提供座位管理、预约管理和统计查看等功能
