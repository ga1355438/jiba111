## 3. 骨架搭建过程

### 3.1 创建 Solution 和 Project

```bash
dotnet new sln -n LibrarySeatReservation
dotnet new mvc -n LibrarySeatReservation.Web -o src/LibrarySeatReservation.Web --framework net6.0
dotnet sln add src/LibrarySeatReservation.Web/LibrarySeatReservation.Web.csproj
```

**说明**：.NET 10 SDK 不直接支持 net8.0 模板，先用 net6.0 创建后手动改为 net8.0。

### 3.2 修改 TargetFramework

将 `LibrarySeatReservation.Web.csproj` 中的 `TargetFramework` 从 `net6.0` 改为 `net8.0`，并添加 EF Core 包：

```xml
<PackageReference Include="Microsoft.EntityFrameworkCore.SqlServer" Version="8.0.0" />
<PackageReference Include="Microsoft.EntityFrameworkCore.Tools" Version="8.0.0" />
```

### 3.3 建立目录结构

| 目录 | 用途 |
|------|------|
| Models/Entities/ | 实体类（Seat, Reservation, AdminUser） |
| Models/Enums/ | 枚举定义（SeatStatus, ReservationStatus） |
| Data/ | DbContext + SeedData |
| DataAccess/ | 数据访问接口与实现 |
| Services/ | 业务逻辑接口与实现 |
| Helpers/ | PasswordHelper |
| Constants/ | TimeSlots |
| Controllers/ | 4 个 Controller |
| Views/ | 9 个页面视图 |

### 3.4 建立 Entity 初稿

- `Seat.cs`：6 个字段，Data Annotations 约束
- `Reservation.cs`：7 个字段，含导航属性 Seat
- `AdminUser.cs`：4 个字段

### 3.5 建立 DbContext

`AppDbContext.cs` 配置了：
- 3 个 DbSet
- 唯一索引（Seat.Name, AdminUser.Username, Reservation 复合索引）
- 外键关系（Reservation.SeatId → Seat.Id）

### 3.6 建立 DataAccess/Service 接口与实现

| 接口 | 实现 | 说明 |
|------|------|------|
| ISeatRepository | SeatRepository | 座位 CRUD |
| IReservationRepository | ReservationRepository | 预约 CRUD + 统计查询 |
| IAdminUserRepository | AdminUserRepository | 管理员查询 |
| ISeatService | SeatService | 座位业务逻辑 |
| IReservationService | ReservationService | 预约业务逻辑 |
| IAdminService | AdminService | 管理员认证 |

### 3.7 配置 Program.cs

- 注册 DbContext（SQL Server LocalDB）
- 注册 Session（30 分钟超时）
- 注册 DataAccess 和 Service 的 DI
- 启动时自动 `Migrate()` + `SeedData.Initialize()`

### 3.8 创建 EF Core 迁移

```bash
dotnet ef migrations add InitialCreate
dotnet ef database update
```

建库建表成功，3 张表 + 全部索引就绪。
