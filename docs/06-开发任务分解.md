## 6. 技术实现要点

### 6.1 数据库连接配置

```json
// appsettings.json
{
  "ConnectionStrings": {
    "DefaultConnection": "Server=(localdb)\\mssqllocaldb;Database=LibrarySeatReservation;Trusted_Connection=True;MultipleActiveResultSets=true"
  }
}
```

### 6.2 Session配置

```csharp
// Program.cs
builder.Services.AddSession(options =>
{
    options.IdleTimeout = TimeSpan.FromMinutes(30);
    options.Cookie.HttpOnly = true;
    options.Cookie.IsEssential = true;
});
```

### 6.3 体验账号Session管理

```csharp
// 切换账号
HttpContext.Session.SetString("UserName", userName);

// 获取当前用户
var currentUser = HttpContext.Session.GetString("UserName") ?? "张三";
```

### 6.4 管理员认证

```csharp
// 检查登录状态
[AdminAuth]
public class AdminController : Controller
{
    // ...
}

// AdminAuth特性
public class AdminAuthAttribute : ActionFilterAttribute
{
    public override void OnActionExecuting(ActionExecutingContext context)
    {
        if (context.HttpContext.Session.GetString("AdminUsername") == null)
        {
            context.Result = new RedirectToActionResult("Login", "Admin", null);
        }
    }
}
```
