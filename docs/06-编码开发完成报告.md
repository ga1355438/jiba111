## 3. 修复记录

| 问题 | 修复方式 |
|------|----------|
| `/Reservation/Create/1` 返回 404 | 添加 `[HttpGet("Reservation/Create/{seatId:int}")]` 路由属性 |
| `/Admin/Seat/Index` 返回 404 | 添加 `[HttpGet("Admin/Seat/Index")]` 路由属性 |
| `/Admin/Reservation/Index` 返回 404 | 添加 `[HttpGet("Admin/Reservation/Index")]` 路由属性 |
| 预约提交时 Reservation.Id 被错误绑定 | 在 `Reservation.Id` 上添加 `[BindNever]` |
