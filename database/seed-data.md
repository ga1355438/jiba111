## 初始化逻辑

```csharp
// SeedData.Initialize 方法
if (context.Seats.Any()) return;  // 已有数据则跳过
// 插入 20 条座位 + 1 条管理员
context.SaveChanges();
```

**注意**：如果Seats表已有数据，种子数据不会重复插入。需先 `dotnet ef database drop` 再重启方可重新初始化。
